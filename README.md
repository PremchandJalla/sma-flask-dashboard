# SMA Flask Dashboard

An interactive dashboard built with Flask and Dash for analyzing and visualizing data insights.

## Features
- Upload `.zip` files containing `.stm` files for data processing.
- Automatically generates insights and displays interactive dashboards.
- Utilizes Nvidia Mistral for AI-generated insights.
- Styled to resemble the Star Micronics branding for a project implementation idea.
